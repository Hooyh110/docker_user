## 添加新文档链接

编写完一个新文档后，修改以下两处。

- `README.md` : 在相应分组下添加新文档链接。

* `.vuepress\config.js` : 在节点 `themeConfig.nav` 相应分组下添加新文档配置，添加右上角导航栏链接。
